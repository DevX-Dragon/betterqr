What it does

Everything you'd want
from a QR library.

Built from the ground up in pure Python — no external QR engine — with a clean, chainable API.

🎨

8 Module Shapes

Square, circle, rounded, diamond, star, gapped, vertical bar, and horizontal bar — mix and match to match your brand.

🌈

Gradient Fills

Apply linear or radial gradients at any angle to make QR codes that feel designed, not generated.

🖼️

Logo Embedding

Drop any PNG or JPG into the center. BetterQR handles clipping, padding, and shape masks automatically.

✨

Animated GIFs

Shimmer, fade, scan, pulse, matrix, wave — six animation effects for QR codes that move.

🖼️

Frames & Labels

Add simple, rounded, double, or shadow frames and text labels above or below — no Photoshop required.

📱

Rich Data Types

WiFi, vCard, MeCard, GeoLocation, Calendar Events, SMS, Email, Phone, and Crypto — all with typed helper classes.

🔲

MicroQR Support

Generate compact MicroQR codes for small-label use cases where every pixel counts.

💻

CLI Included

The betterqr command ships with the package — generate QR codes without writing a line of code.

📦

Multiple Formats

Export as PNG, JPG, SVG, or GIF. Transparent backgrounds supported. JPEG quality control built in.

Python API

Fluent by design.

Chain methods to build exactly what you want — no boilerplate, no config files.

from betterqr import QR

# Simplest possible usage
QR("https://example.com").save("qr.png")

# With custom settings
qr = QR(
    data="https://example.com",
    ecc="H",       # Error correction level
    box_size=10,
    border=4
)
qr.save("qr_custom.png")

# Display in terminal
qr.display()
from betterqr import QR

# Colored + shaped
QR("https://example.com").style(
    fill_color="#6C3082",
    back_color="#F3E8FF",
    shape="circle",
    radius=0.8
).save("styled.png")

# Gradient fill
QR("Hello Gradient").gradient(
    start_color="#FF6B6B",
    end_color="#4ECDC4",
    direction="radial"
).save("gradient.png")

# Transparent background
QR("https://example.com").style(
    back_color="transparent"
).save("transparent.png")
from betterqr import QR

# Embed a logo in the center
# Use ecc="H" for best scannability
QR("https://mysite.com", ecc="H").logo(
    image_path="logo.png",
    ratio=0.3,          # 0.2–0.4 recommended
    shape="rounded",
    padding=0.05
).save("logo_qr.png")

# Add a frame + label too
QR("https://mysite.com", ecc="H")\
    .logo("logo.png")\
    .frame(style="shadow", color="#333")\
    .label(text="Scan me", position="bottom")\
    .save("branded.png")
from betterqr import QR

# Matrix rain effect
QR("Animated QR").animate(
    effect="matrix",
    frames=30,
    fps=12
).save("matrix.gif")

# All available effects:
# shimmer · fade · scan
# pulse · matrix · wave

# With duration control
QR("Hello").animate(
    effect="pulse",
    duration=3.0,
    fps=20
).save("pulse.gif")
from betterqr import QR, WiFi, VCard, SMS, Email

# WiFi QR code
QR(WiFi("MyNet", "password", security="WPA"))\
    .save("wifi.png")

# Contact card
QR(VCard(
    name="Jane Doe",
    phone="+1-555-1234",
    email="jane@example.com",
    organization="Acme Corp"
)).save("contact.png")

Chainable. Readable. Pythonic.

Every method returns the QR object, so you can chain style, logo, frame, label, animate, and save — in any order that makes sense for your use case.

  • Zero external QR dependencies
  • Python 3.10–3.13 supported
  • Only dependency: Pillow ≥ 10.0
  • Auto-detects version and ECC
  • PIL Image access via .get_image()
  • Metadata: .version, .size, .data_capacity
Data Types

Not just URLs.

Dedicated helper classes format the data string correctly for every QR standard — so you don't have to.

📶

WiFi

WPA, WEP, or open networks. Includes hidden network support.

👤

VCard

Full contact cards with name, phone, email, org, URL, and address.

📇

MeCard

Compact contact format for older device compatibility.

📍

GeoLocation

Lat/lon coordinates, optional altitude and location query.

📅

Event

Calendar events with title, start/end time, location, description.

💬

SMS

Pre-filled SMS to a phone number with an optional message body.

📧

Email

Email with address, subject line, and body pre-populated.

📞

Phone

Tap-to-call QR codes for any phone number format.

₿

Crypto

Bitcoin, Ethereum, and more — with address and optional amount.

Command Line

No code required.

The betterqr CLI ships with the package. Full feature parity with the Python API.

betterqr "https://example.com"

Generate a basic QR code (outputs qr.png)

betterqr "https://mysite.com" out.png \
  --logo logo.png --ecc H \
  --shape circle --fill "#6C3082"

Styled QR with a logo

betterqr "Hello" animated.gif \
  --effect matrix --fps 12

Animated GIF with matrix effect

betterqr "https://example.com" --display

Preview directly in your terminal

FlagDescription
--eccError correction (L/M/Q/H)
--shapeModule shape (8 options)
--fillFill color (hex)
--backBackground color (hex)
--gradient-start/endGradient colors
--logoPath to logo image
--logo-ratioLogo size (0–0.4)
--frameFrame style
--labelText label
--effectAnimation effect
--fpsAnimation frame rate
--formatpng / jpg / svg / gif
--displayPrint to terminal

Installation

Up and running in seconds.

1

Install via pip

BetterQR is on PyPI. Python 3.10 or higher required.

pip install betterqr
2

Import and use

The QR class is the single entry point for everything.

from betterqr import QR
3

Or use the CLI

The betterqr command is available immediately after install.

betterqr "your text"
4

Install from source

For contributors and edge builds — clone and install editable.

pip install -e .
